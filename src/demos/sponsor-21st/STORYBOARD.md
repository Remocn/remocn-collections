# STORYBOARD — sponsor-21st

Sponsor spot for **21st.dev** — "The living library of interfaces." Sibling of
`sponsor-reui` / `sponsor-reactbits`: same sponsor arc, 21st's register (Geist,
near-black monochrome chrome, periwinkle-indigo shader). The heart is the **gallery**
— a stream of gorgeous, distinct interfaces flowing past (the reference clip's
language), because with 21st the UI *is* the value.

**Product truth**
- **Audience:** developers & design engineers who build React/Tailwind UI — and, now, their AI agents.
- **Pain / desire:** building beautiful UI from scratch is slow; you want a huge, high-craft library you can drop straight in.
- **Promise:** a *living* library of 10,000 real, hand-crafted interfaces — install any one with a single command, for you and your agent.
- **Product role:** the gallery you (and your agent) pull polished, production-grade components from.
- **Proof:** 10,000+ components · built by real design engineers · 727,000 builders · Google/Meta/NVIDIA/Vercel/Lovable · live preview · one-command install · agent-ready.
- **CTA:** 21st.dev — browse the living library.

**Transition legend** (registry name → remocn factory): `zoom-through → pushThrough`
(section turns) · `blur-crossfade → focusPull` (gallery beats) · `crossfade → crossfade`.
No swirl / ripple. The shader field rides under every frame with a slow drift.
